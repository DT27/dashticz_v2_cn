# dashticz_v2_cn
## Dashticz中国版
下载后将dashticz整个文件夹放到 `domoticz/www/`目录中 
例如 `domoticz/www/dash/`
### 访问方法
如果domoticz地址是 http://192.168.1.123:8080/  
那么dashticz地址就是 http://192.168.1.123:8080/dash/ 
不需要设置domoticz服务器地址。
